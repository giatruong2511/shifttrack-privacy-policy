Privacy Policy – ShiftTrack

Effective date: 2026-01-01

ShiftTrack respects your privacy. This application does not collect, store, transmit, or share any personal data with external servers.

1. Data Collection

ShiftTrack stores all data locally on the user’s device only, including:

  • Employee lists
  • Shift checklists
  • Work schedules
  • Admin password (encrypted locally)

No personal information such as name, email, phone number, location, photos, or biometric data is collected.

2. Data Usage

Data is used solely to support the core functionality of the application and never leaves the device.

3. Data Sharing

ShiftTrack does not share any data with third parties.

4. Data Security

All information is stored locally using secure storage mechanisms. Admin passwords are encrypted and can be changed by the user at any time.

5. User Rights

Users can delete all data by uninstalling the application.

6. Children

ShiftTrack is not intended for children under 13 years of age.

This policy may be updated in future versions of the application.
